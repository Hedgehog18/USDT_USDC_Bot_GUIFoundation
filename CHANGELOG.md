# Changelog

## GUI MVP Checkpoint

- Dashboard summary for system status, database counters, latest backtest, latest paper run, safety events, and notifications.
- Health tab for config, SQLite, and market-data checks.
- Runner monitor/control with guarded Demo Runner Start/Stop, QThread execution, status states, last-run summary, and system event logging.
- Backtest GUI controls for interval/limit execution, result storage, and report generation.
- Paper Trading GUI controls for paper simulation, result display, and data refresh.
- Analytics charts for latest backtest equity curve, drawdown curve, and trade PnL distribution.
- Logs diagnostics for file logs and SQLite system events with filters and auto-refresh.
- Settings read-only tab for current configuration visibility.

Real trading remains disabled.
